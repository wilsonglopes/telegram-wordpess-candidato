Como instalar o plugin CampanhaPress
⬇️ Baixar CampanhaPress.zip

  1. Clique em Baixar CampanhaPress.zip acima para obter o arquivo do plugin.

  2. No seu WordPress, acesse Plugins → Adicionar novo → Enviar plugin.

  3. Selecione o arquivo campanhapress.zip baixado e clique em Instalar agora.

  4. Clique em Ativar plugin.

  5. No menu lateral do WordPress, clique em CampanhaPress.

  6. Copie a Chave de API exibida (começa com letras/números, ~40 caracteres).

  7. Cole a chave no campo "Chave do Plugin CampanhaPress" acima e clique em Salvar WordPress.

⚠️ A chave de API não deve ser compartilhada. Se suspeitar de uso indevido, clique em "Regenerar chave" no painel WordPress e atualize aqui.

Como obter o Token do Facebook e ID do Instagram (passo a passo completo)

Parte 1 — Criar o aplicativo Facebook

  1. Acesse developers.facebook.com e faça login com a conta pessoal do Facebook vinculada à sua Página.

  2. Clique em Meus Apps → Criar App. Escolha o caso de uso "Gerenciamento de conteúdo" e clique em Avançar.

  3. Dê um nome ao aplicativo (qualquer nome serve) e clique em Criar app.

Parte 2 — Configurar permissões do Instagram

  1. Dentro do app criado, clique em Casos de uso no menu lateral esquerdo.

  2. Localize "Gerenciar mensagens e conteúdo do Instagram" e clique em Personalizar.

  3. Clique em Permissões e recursos. Vai abrir a lista de permissões disponíveis.

  4. Encontre instagram_basic e clique no botão + ao lado para adicionar.

  5. Encontre instagram_content_publishing e clique no botão + para adicionar também.

  6. Clique em Voltar para retornar à tela de Casos de uso.

Parte 3 — Configurar permissões do Facebook

  1. Ainda em Casos de uso, localize "Gerenciar tudo na sua página" e clique em Personalizar.

  2. Clique em Permissões e recursos.

  3. Encontre pages_manage_posts e clique em + para adicionar.

  4. Encontre pages_read_engagement e clique em + para adicionar.

  5. A permissão pages_show_list provavelmente já está marcada — se não estiver, adicione também.

  6. Clique em Voltar.

Parte 4 — Gerar o token inicial no Explorador de API

  1. No menu superior do Facebook for Developers, clique em Ferramentas → Explorador de API do Gráfico.

  2. No campo "Usuário ou Página", clique e selecione "Acesso a uma Página". Se a sua página não aparecer, clique em "Adicionar uma página" e autorize.

  3. Clique em Gerar token. O Facebook vai pedir login e aprovação das permissões — autorize tudo.

  4. Um token vai aparecer no campo "Token de acesso". Copie esse token.

Parte 5 — Estender o token para longa duração

  1. Abra uma nova aba no navegador e acesse o Depurador de Token de Acesso: developers.facebook.com/tools/debug

  2. Cole o token copiado no campo de texto e clique em Depurar.

  3. Verifique o campo "Expires" — o token provavelmente expira em poucas horas.

  4. Role a página até o final e clique em "Estender token de acesso". O Facebook vai pedir autenticação novamente.

  5. Após autenticar, um novo token aparecerá em verde na tela. Copie esse novo token.

Parte 6 — Obter o token permanente da Página

  1. Volte para a aba do Explorador de API do Gráfico (que está aberta desde a Parte 4).

  2. No campo "Token de acesso", apague o token antigo e cole o token verde obtido na Parte 5.

  3. No campo de endpoint (a barra ao lado do botão Enviar), apague o que estiver escrito e digite exatamente: me/accounts?fields=name,access_token

  4. Clique em Enviar. Vai aparecer uma resposta com o nome da sua Página e um campo access_token.

  5. Copie o valor do campo access_token da resposta — esse é o token permanente da Página, que não expira.

  6. Cole esse token no campo "Token de Acesso da Página" acima e clique em Salvar Facebook & Instagram.

Parte 7 — Obter o ID do Instagram Business

  1. Ainda no Explorador de API, com o token permanente colado, no campo de endpoint digite: me?fields=instagram_business_account

  2. Clique em Enviar. A resposta trará um campo instagram_business_account com um id.

  3. Copie esse id e cole no campo "ID do Usuário Instagram Business" acima.

  4. Clique em Salvar Facebook & Instagram.

⚠️ Atenção: o token da Parte 4 expira em horas — só o token obtido na Parte 6 (via me/accounts) é permanente e deve ser usado aqui. Se o Facebook parar de publicar, o primeiro suspeito é sempre um token expirado.

Grupos de distribuição

Como conectar o WhatsApp

  1. O QR code expira em ~40 segundos. Se expirar, clique em "Gerar novo QR".

  2. No celular da campanha, abra o WhatsApp e toque nos três pontos (⋮).

  3. Selecione Aparelhos conectados → Conectar um aparelho.

  4. Aponte a câmera para o QR code. O status mudará para Conectado ✅ automaticamente.

  5. Após conectar, clique em Sincronizar grupos para carregar os grupos disponíveis.

  6. Ative os grupos desejados clicando no toggle ao lado de cada um.

⚠️ O número conectado precisa estar dentro dos grupos para que eles apareçam na lista.
